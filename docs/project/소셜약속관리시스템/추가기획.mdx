

### **재미 기능: 친근하고 웃긴 계약서**

#### 1. **각서 스타일 계약서**
- **설명**: 친구들 간의 약속을 좀 더 재미있고 유쾌하게 기록하는 각서 스타일의 계약서. 예를 들어, "절대 잊어서는 안 될 각서"나 "안 지키면 혼날 각서" 같은 유머러스한 표현을 사용할 수 있습니다.

- **기능**:
    - **계약서 스타일 선택**: 사용자가 약속을 기록할 때, 진지한 계약서와 **유쾌한 각서** 중에서 선택할 수 있는 옵션을 제공합니다.
    - **유머러스한 템플릿**: 다양한 웃긴 계약서 템플릿을 제공합니다. 예를 들어, "이걸 안 지키면 평생 라면만 먹겠다" 같은 약속 내용을 자동으로 추가하는 기능.
    - **친구 이름이나 약속 내용에 맞춘 맞춤형 텍스트**: "친구 A가 이걸 지키지 않으면 B는 매일 아침 6시에 전화해서 깨울 권리를 가진다!" 같은 맞춤형 약속을 만들어줍니다.

#### 2. **랜덤 각서 생성 기능**
- **설명**: 사용자가 약속을 기록할 때, "랜덤 각서" 기능을 켜면 자동으로 재미있는 벌칙이 포함된 각서를 생성해줍니다.

- **기능**:
    - **자동 생성된 벌칙**: 사용자가 약속을 지키지 않으면 발생할 황당하고 재미있는 벌칙을 자동으로 생성합니다. 예를 들어, "약속을 지키지 않으면 친구들에게 매주 새로운 농담을 보내야 한다" 같은 벌칙을 제안.
    - **사용자 맞춤형 벌칙 설정**: 사용자가 벌칙을 직접 입력할 수 있도록 맞춤형 벌칙 기능을 제공하여 개인적인 재미를 더할 수 있게 함.
    - **벌칙 이행 추적**: 약속을 지키지 않았을 때 벌칙을 어떻게 이행했는지 친구들과 함께 추적할 수 있는 간단한 알림 기능.

#### 3. **계약서 커스터마이징 기능**
- **설명**: 계약서의 텍스트와 이미지를 사용자가 직접 커스터마이징할 수 있어, 각기 다른 상황에 맞는 웃긴 계약서를 생성할 수 있습니다.

- **기능**:
    - **이모티콘 및 이미지 추가**: 계약서에 이모티콘이나 웃긴 이미지를 추가할 수 있어 더 유쾌한 분위기를 조성.
    - **계약서 제목 변경**: 사용자가 계약서 제목을 "절대 지켜야 할 비장한 각서", "우리는 이걸 지키지 않으면 안 돼!" 등으로 자유롭게 바꿀 수 있게 함.
    - **계약서 배경**: 계약서 배경을 재미있고 다채로운 이미지나 색상으로 변경하여, 사용자가 원하는 스타일로 꾸밀 수 있게 함.

#### 4. **약속 이행 시 보상 시스템**
- **설명**: 약속을 잘 지킨 사용자에게 소소한 보상을 주는 시스템을 추가하여 약속을 즐겁게 지킬 수 있도록 동기 부여.

- **기능**:
    - **보상 배지**: 약속을 이행한 사용자에게 "최고의 약속 이행자", "친구의 신뢰를 저버리지 않은 영웅" 같은 재미있는 배지를 부여.
    - **약속 이행 통계**: 친구들 간의 약속 이행 통계를 모아서 누가 더 많은 약속을 지켰는지 순위표 제공.
    - **달성 포인트 시스템**: 약속을 지킬 때마다 포인트를 얻고, 이 포인트를 누적하여 가상의 보상을 받을 수 있도록 설정. 예를 들어, 포인트가 모이면 앱 내에서 사용할 수 있는 재미있는 아이템(이모티콘, 프로필 배경 등)을 제공.

#### 5. **친구끼리 내기하기**
- **설명**: 친구들 간에 약속을 하면서 소소한 내기를 추가하는 기능.

- **기능**:
    - **내기 추가 기능**: 약속을 기록할 때, "내기" 옵션을 추가해 누가 더 빨리 약속을 이행하는지, 또는 더 잘 지키는지를 게임처럼 경쟁.
    - **승자 보상**: 내기에서 이긴 사람에게 특별한 보상(가상 트로피, 배지 등)을 부여하고, 내기 이행 상황을 추적.
    - **벌칙**: 내기에서 진 사람이 수행해야 할 벌칙을 기록하고 추적할 수 있는 기능.

#### 6. **소셜 공유 기능**
- **설명**: 재미있는 계약서나 벌칙 내용을 카카오톡, 인스타그램 등 소셜 미디어에서 쉽게 공유할 수 있는 기능을 추가하여 사용자들끼리 더 많은 약속을 공유하고 자랑할 수 있도록 함.

- **기능**:
    - **계약서 이미지로 변환 및 공유**: 사용자가 만든 계약서나 약속 내용을 이미지로 변환해 소셜 미디어에서 공유.
    - **웃긴 벌칙 모음 공유**: 사용자가 약속을 지키지 않았을 때 생성된 웃긴 벌칙들을 공유할 수 있어, 친구들끼리의 재미를 극대화.

---

### **기능 요약**
- **각서 스타일 계약서**: 유쾌하고 웃긴 계약서 템플릿 제공.
- **랜덤 각서 생성**: 랜덤으로 재미있는 벌칙을 포함한 각서 자동 생성.
- **계약서 커스터마이징**: 사용자가 계약서의 제목, 내용, 이미지 등을 자유롭게 변경하고 꾸밀 수 있는 기능.
- **보상 및 벌칙 시스템**: 약속을 이행했을 때 보상을 주고, 지키지 않았을 때 재미있는 벌칙을 수행.
- **내기 기능**: 친구들 간의 약속을 내기처럼 게임화하여 승자를 가림.
- **소셜 공유 기능**: 재미있는 계약서나 벌칙 내용을 카카오톡, 인스타그램 등 소셜 미디어에서 공유.

### **기대 효과**
- 약속을 기록하는 진지한 시스템을 넘어서, **재미 요소**를 가미하여 친구들끼리 자주 사용하게끔 유도.
- 사용자들이 약속을 더 잘 지키도록 **동기 부여**.
- **소셜 공유** 기능을 통해 앱의 바이럴 마케팅 효과를 얻어 더 많은 사용자 유입 기대.
