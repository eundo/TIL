# Eundo Today Portfolio

이 사이트는 기존 TIL 블로그를 포트폴리오 중심 구조로 재정리한 Docusaurus 프로젝트입니다. About Me, Case Studies, Projects, Notes의 역할을 나누고 GitHub push 기반 Netlify 배포 흐름을 사용합니다.

## Snapshot

| 구분 | 내용 |
| --- | --- |
| 상태 | Public portfolio site |
| 문제 | TIL, 회고, 프로젝트 기록이 섞여 있어 포트폴리오 관점의 첫인상이 약함 |
| 접근 | 첫 화면과 메뉴 구조를 포트폴리오 중심으로 재배치하고, 프로젝트 허브를 별도 설계 |
| 기술 | Docusaurus, TypeScript, MDX, CSS Modules, Netlify |
| 배포 | `main` 브랜치 push 후 Netlify에서 정적 빌드 및 배포 |

## What It Does

- 개발자 소개, 실무 사례, 프로젝트, 학습 노트를 목적별 메뉴로 분리합니다.
- 실무 문제 해결 사례는 Case Studies로, 제품형 개인 실험은 Projects로 분리합니다.
- MDX 기반으로 프로젝트 상세 페이지를 계속 확장할 수 있습니다.
- Netlify 설정을 repository에 고정해 배포 재현성을 높였습니다.

## Portfolio Signal

이 프로젝트는 결과물 자체가 포트폴리오이면서, 문서화와 배포 자동화까지 포함한 작은 운영 시스템입니다. 단순한 정적 페이지가 아니라 콘텐츠 구조, 경력 소개, 프로젝트 정리 방식까지 함께 개선하고 있습니다.
