# Projects

실무에서 다뤄온 운영 시스템 경험과 개인적으로 제품화해 보고 있는 도구를 함께 정리합니다. 진행 중인 private repository는 코드 링크 대신 문제 정의, 구현 범위, 기술 선택, 다음 개선 방향만 공개합니다.

:::note
일부 프로젝트는 아직 개발 중인 private repository입니다. 공개 페이지에는 핵심 아이디어를 그대로 복제할 수 있는 세부 구현이나 내부 코드를 노출하지 않습니다.
:::

## 정리 기준

- **문제:** 어떤 반복 작업, 사용성 문제, 운영상의 비효율을 해결하려 했는지
- **역할:** 직접 설계하고 구현한 범위와 의사결정
- **접근:** 기술 선택, 데이터 흐름, 배포 방식에서의 판단
- **공개 범위:** 진행 중인 private 프로젝트는 코드 링크 없이 안전한 수준의 구조와 배운 점만 공개
- **다음 단계:** 제품 완성도를 높이기 위해 남겨둔 개선 항목

## 보류한 항목
