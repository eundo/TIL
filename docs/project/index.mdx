## 정리 기준

- **문제:** 어떤 사용자/운영 문제가 있었는지
- **역할:** 내가 맡은 설계, 개발, 운영 범위
- **접근:** 어떤 기술 선택과 구조 판단을 했는지
- **결과:** 성능, 안정성, 운영 효율 측면에서 무엇이 나아졌는지
