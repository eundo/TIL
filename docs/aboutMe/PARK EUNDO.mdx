## 대표 개선 경험

### React 기반 웹앱 SPA 전환

LG CNS 마이데이터 플랫폼에서 팝업 기능 때문에 일부 MPA 구조로 동작하던 React 웹앱을 SPA 구조로 개선했습니다.

- 기존 MPA 구조 분석 및 SPA 전환 계획 수립
- `react-activation`을 적용해 팝업/페이지 상태 유지
- Redux 기반 전역 상태 관리
- 필요한 CSS만 로드하도록 최적화
- 페이지 전환 속도와 브라우저 내비게이션 경험 개선

### 보험금 지급 오류 재이체 처리 시스템

흥국생명 IT 운영 중 보험금 지급 오류 건을 수작업 DB 변경 없이 재처리할 수 있는 화면과 프로세스를 개발했습니다.

- 재이체 대상 오류 건 자동 조회
- 재이체 불가 오류와 가능 오류를 구분하는 조건 설계
- 처리자 세션과 지급/이체 상태를 단계별로 초기화하는 흐름 구현
- 반복 오류 처리 시간을 단축하고 현업의 수작업 부담 감소

### 고객 히스토리 조회 시스템

고객 문의와 컴플레인 대응을 위해 로그를 직접 뒤지던 방식을 개선하고, 고객별 이력을 조회할 수 있는 화면을 개발했습니다.

- 주요 서비스에 공통 로깅 프리픽스 적용
- 서비스명/액션별 히스토리 구분
